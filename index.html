Web Revival: What is it?

The web revival is a movement focused on reclaiming technology and reviving the spirit of the internet in its early days. It involves movements such as the indie web, old web, and small web.

Some older sites that these movements draw inspiration from are Geocities and MySpace.

HTML

What is HTML?

HTML stands for Hypertext Markup Language. It is used to describe the structure of a webpage.
HTML consists of a series of elements, which tell browsers what to display on a web page.
An element is made up of an opening tag, content, and a closing tag.
<opening tag> Content <closing tag>

CSS is what we use to style a HTML document. We will come back to this later!

Javascript is the language used to make webpages do things.

HTML Editors

Structure of a HTML Document

Before we write any HTML, we're going to look at some! To view the source code of a webpage, press ctrl + u

HTML does not need indentation to work, but it makes it much easier to see where everything is.

<!DOCTYPE html> this defines the document type

<html> all your code will go inside the html tag

<head> contains meta information about the webpage
<title> the title of the webpage (the name shown in the tab)

<body>

the <body> tag is where all the contet for your site goes. everything in here will be displayed on your webpage.

Headings: <h1> to <h6>
Paragraph: <p>
Links: <a>
Images: <img>
Lists: <ul> for an unordered list (bullet points), and <ol> for a numbered list. list items are added with <li>
<span> can be used to create a span of code that is styled differently
<div> is used to separate code into 'sections', which are incredibly useful for styling and creating layouts.

Empty elements are HTML elements that do not need closing tags. <img> and <br> (line break) are empty elements

Block level vs. inline elements:
Block level elements start on a new line. <p> and <div>
Inline elements do not start on a new line. <a> and <b> (bold text).

CSS

CSS stands for Cascading Style Sheets.

Here are some cool examples of how CSS can be used to style a webpage:

There are multiple ways of adding CSS to a page: inline, internal, and external.

Inline: style is defined inside of a html tag
<p style='color:red;'>Red Text</p>
Output:

Red Text

Internal: CSS is used in the html <head> tag, by adding <style></style>

External: CSS is written in a separate 'style.css' document and linked in the head tag with
<link rel='stylesheet' href='style.css'>

CSS Format

element {
    property:value;
    property:value;
}

CSS Properties

Color: sets the text colour (note that CSS uses american spellings!)
Background: sets the background colour
Background-image: sets a background image
Font-family: sets a Font-family
Height & Width: sets an element's size

My Advice

Be curious, adventurous, and ambitious!

The best way to learn how something works is to break it & put it back together.

some good habits:

  • adding target='_blank' to a link will make it open in a new tab
  • keep your files organised
  • don't use spaces in file or folder names
  • leave mesages in people's guestbooks :)
  • focus on adding content to your site & don't worry yourself about creating a layout immediately
  • a lot of people will ask you not to hotlink images from their site. this means you should download any images you want to use & host them yourself, rather than linking their website.
  • if you want to copy & paste large chunks of code, instead re-write them line by line so you can see how it works
  • your website is yours!
